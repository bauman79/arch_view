DXF를 불러오거나 라이브러리에서 건물을 추가하세요.

대지 위치·격자

위도·경도·분석일(툴바)·격자는 그림자 미리보기, 일조권 검토, PV 분석이 공용으로 씁니다.

정북·채광·인동 기준

기본값 = 건축법 시행령 제86조. 정북사선은 항상 자동 검토, 채광·인동은 토글을 켜면 이동·회전 중 실시간 갱신. PLAN_WIN 레이어로 창면 표시 시 완화 기준 적용.

일조권 검토 (수인한도)

대법원 판례 기준 — 인접건물이 기준일 09~15시 연속 2시간 또는 08~16시 총 4시간 이상 직달일사를 받는지 판정. 정북사선(거리 기준)과는 별개.

히트맵: 파랑=적합, 빨강=위반

PV 상대평가 (M4)

대표일 4일(춘분·하지·추분·동지) 8~16시 직달 누적 × 입사각 보정 — 인접건물 전체 벽면·지붕

히트맵: 파랑=낮음 → 노랑 → 빨강=높음 (최대값 대비 %)

PV 절대평가 (M5 · TMY)

EPW 8,760시간: 직달(DNI×cosθ, 차폐 레이캐스팅) + 산란(DHI) → 연간 kWh/㎡

히트맵: 파랑=0 → 노랑 → 빨강=최대 kWh/㎡

바람길 분석 (M8)

EPW 풍향·풍속 통계의 주풍향으로 2D 포텐셜 흐름을 근사해 스트림라인과 바람그림자(정체 영역)를 표시 — CFD가 아닌 개략 검토용.

스트림라인: 파랑=정체(<0.3배) → 청록 → 초록(≈1배) → 주황 → 빨강(>2배) — 주풍속 대비 배율

LBM 정밀 시뮬레이션 (M10) — D2Q9 격자 볼츠만 2D 유동. 수렴(연속 10스텝 변화율 <0.1%)하면 자동 정지. 위의 포텐셜 근사와 별개 모드이며, 수직 흐름·난류 상세는 범위 밖.

히트맵: 파랑=정체 → 청록 → 노랑(≈주풍속) → 빨강(2배 이상) — U/U₀. 유선 색은 M8과 동일 규약. 풍향 자동은 위 기상데이터·기간 선택을 따름.

일조시간 지도 (M9)

지면·전체 건물 표면(계획주동 + 인접건물 입면·지붕)의 기준일 08~16시 직달일조 시간 히트맵(30분 간격, 격자 2m) — 계획주동이 인접건물 일조를 얼마나 깎는지 확인. 일조권 검토(수인한도 판정)와는 별개의 시각화 도구.

0h=암청 → 1h=남색 → 2h=파랑 → 4h=청록 → 6h=초록 → 8h=노랑

지형 (M7)

DXF CONTOUR 레이어 등고선으로 TIN 지형을 표시하고 건물을 지반고(G.L.)에 얹습니다. 시각화 전용 — 정북·채광·인동 검토는 지형과 무관하게 동일 레벨 기준. 규약은 툴바 "레이어 규약" 참고.

계단식을 켜면 지형이 등고 단으로 표시되고 주동 지반고(G.L.)도 단 레벨로 스냅되어 단 위에 앉습니다(끄면 TIN 보간 복귀). 정북·채광·인동 검토는 어느 모드든 동일 레벨 기준.

DXF 레이어 규약

주동 레이어(권장): PLAN_<타입>_<번호> — 예: PLAN_A_1, PLAN_B_2 (타입 문자는 타워형·판상형 등 임의 구분, 층수는 기본 15층 — 목록에서 조정). 인접건물: ADJ_BLDG_<n>F 또는 ADJ_<타입>_<번호>. 구버전 PLAN_BLDG_<n>F도 계속 인식됩니다. 경계선: SITE_BOUNDARY(대지) / ADJ_BOUNDARY(인접대지 — 정북·채광사선 기준선) / ROAD_CL(도로중심선) / PARK_BOUNDARY(공원). 창면: PLAN_WIN(footprint 에지와 같은 좌표에 그리기). 단위는 $INSUNITS 자동 인식(없으면 mm 가정). 여러 파일로 나눠 그렸으면 먼저 하나를 불러온 뒤 "DXF 추가"로 병합.
